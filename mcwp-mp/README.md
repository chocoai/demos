## 前言
>  本工程主要基于 mpvue 实现。

>  使用技术： mpvue

>  项目说明： 本工程主要基于 mpvue 实现。

### 下载

```bash
# git clone

git clone git@qdjr.git.zhudb.com:zdb/mcwp-web.git

cd mcwp-web
cd mcwp-mp
```

### 安装
``` bash
# 安装前请先确保全局环境下已安装node、npm和yarn
$ yarn bootstrap
```

### 运行
```bash
# 开发模式
$ yarn dev

# 生产模式
$ yarn build
```
